# 사용자 매뉴얼

## 🚀 시작하기

### 1. 로그인
- Google 계정으로 로그인
- 또는 이메일 Magic Link 사용

### 2. 대시보드 살펴보기
- 주요 지표 확인
- 진행 중인 캠페인 보기
- 최근 활동 확인

### 3. 광고주 등록
1. 좌측 메뉴에서 "광고주" 클릭
2. "+ 새 광고주" 버튼 클릭
3. 필수 정보 입력:
   - 회사명
   - 사업자등록번호
   - 담당자 정보
   - 연락처
4. "저장" 버튼 클릭

### 4. 캠페인 생성
1. "캠페인" 메뉴 선택
2. "+ 새 캠페인" 버튼
3. 캠페인 정보 입력:
   - 캠페인명
   - 광고주 선택
   - 기간 설정
   - 예산 입력
   - 목표 설정
4. "생성" 클릭

### 5. 견적서 작성
1. "견적서" 메뉴 이동
2. "+ 새 견적서"
3. 광고주 선택
4. 상품 추가:
   - 상품 검색
   - 수량 입력
   - 단가 확인
5. 총액 확인
6. "견적서 생성"

### 6. 리포트 확인
1. "리포트" 메뉴
2. 캠페인 선택
3. 기간 설정
4. 지표 확인:
   - 노출수
   - 클릭수
   - 전환수
   - CTR, CVR
5. PDF 다운로드 또는 이메일 발송

# 트러블슈팅

## 🔧 일반적인 문제

### 1. 로그인 안됨
**증상**: Google 로그인 후 리다이렉트 실패  
**원인**: OAuth 설정 오류  
**해결**:
```bash
# .env 파일 확인
GOOGLE_CALLBACK_URL 확인
# 서버 재시작
npm run dev
```

### 2. 데이터 로딩 실패
**증상**: 목록 페이지에서 데이터가 안보임  
**원인**: Airtable 연결 오류  
**해결**:
```bash
# Airtable 연결 테스트
npm run test:airtable
# API 키 확인
echo $AIRTABLE_API_KEY
```

### 3. 견적서 PDF 생성 안됨
**증상**: PDF 다운로드 버튼 클릭 시 에러  
**원인**: 서버 메모리 부족 또는 PDF 라이브러리 오류  
**해결**:
```bash
# 서버 로그 확인
# 필요시 서버 재시작
```

### 4. 이메일 발송 실패
**증상**: Magic Link 또는 견적서 이메일 안감  
**원인**: Resend API 키 미설정  
**해결**:
- Resend 설정 확인
- RESEND_API_KEY 환경 변수 설정

### 5. SMS 발송 실패
**증상**: 알림 SMS가 발송되지 않음  
**원인**: Solapi 크레딧 부족 또는 API 오류  
**해결**:
- Solapi 대시보드에서 크레딧 확인
- API 키 유효성 확인

## 🐛 에러 코드

|------|------|----------|
