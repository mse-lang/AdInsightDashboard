# 캘린더 사용 가이드

## 빠른 시작

### 캘린더가 로그인을 요구할 때

캘린더 페이지에서 로그인 화면이 보이면, **"새 탭에서 열기"** 버튼을 클릭하세요!

```
┌─────────────────────────────────────────────────┐
│ ℹ️ 캘린더 보기                                    │
│                                                 │
│ Google Calendar가 iframe에서 로그인을 요구하는   │
│ 경우, 아래 버튼을 클릭하여 새 탭에서 캘린더를     │
│ 열어주세요.                                      │
│                                                 │
│              [🔗 새 탭에서 열기]                  │
└─────────────────────────────────────────────────┘
```

## 3가지 접근 방법

### 1️⃣ 상단 Alert 버튼 (권장)

페이지 상단에 있는 파란색 Alert 박스의 버튼을 클릭하세요.

**장점**:
- 페이지 로드 시 바로 보임
- 가장 빠른 접근

### 2️⃣ 카드 헤더 버튼

캘린더 카드 제목 옆의 버튼을 클릭하세요.

```
┌─────────────────────────────────────────────────┐
│ 월별 광고 일정        [Google Calendar에서 열기] │
├─────────────────────────────────────────────────┤
│                                                 │
│           [캘린더 iframe 영역]                    │
│                                                 │
└─────────────────────────────────────────────────┘
```

**장점**:
- 캘린더 영역에서 바로 접근
- 시각적으로 명확

### 3️⃣ 전체 화면 보기 버튼

페이지 하단의 큰 버튼을 클릭하세요.

```
┌─────────────────────────────────────────────────┐
│ 캘린더 정보                                       │
├─────────────────────────────────────────────────┤
│ 캘린더 ID: mj@venturesquare.net                 │
│                                                 │
│ 💡 Tip: iframe에서 로그인이 요구되는 경우,        │
│ "새 탭에서 열기" 버튼을 사용하면 현재 Google     │
│ 계정으로 자동 로그인됩니다.                       │
│                                                 │
│ ┌─────────────────────────────────────────────┐ │
│ │ 📅 Google Calendar에서 전체 화면으로 보기    │ │
│ └─────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────┘
```

**장점**:
- 전체 화면으로 더 넓게 볼 수 있음
- 캘린더 정보 확인 후 접근

## 왜 새 탭에서 열어야 하나요?

### iframe의 제약

iframe으로 임베드된 Google Calendar는 별도의 보안 컨텍스트에서 작동합니다.

```
VS-AMS (로그인 ✅)
├─ 메인 페이지 (인증됨 ✅)
├─ 캠페인 관리 (인증됨 ✅)
└─ 캘린더 페이지
    └─ iframe (별도 컨텍스트 ⚠️)
        └─ Google Calendar (재로그인 필요 ❌)
```

### 새 탭의 장점

새 탭에서 열면 브라우저의 Google 세션을 그대로 사용합니다.

```
VS-AMS (로그인 ✅)
    ↓
버튼 클릭
    ↓
새 탭 열기
    ↓
브라우저의 Google 세션 사용 ✅
    ↓
캘린더 자동 로그인 ✅
```

## 사용 시나리오

### 시나리오 1: 빠른 확인

```
1. 캘린더 메뉴 클릭
2. 상단 Alert 버튼 클릭
3. 새 탭에서 캘린더 확인
4. 탭 닫기
```

⏱️ **소요 시간**: 5초

### 시나리오 2: 집중 작업

```
1. 캘린더 페이지 하단 스크롤
2. "전체 화면으로 보기" 버튼 클릭
3. 새 탭에서 캘린더 전체 화면 사용
4. 일정 추가/수정 작업
5. 완료 후 VS-AMS 탭으로 복귀
```

⏱️ **소요 시간**: 필요한 만큼

### 시나리오 3: 캠페인 등록 후 확인

```
1. 캠페인 생성 (Google Calendar 자동 동기화)
2. 캘린더 메뉴로 이동
3. 카드 헤더 버튼 클릭
4. 새 탭에서 방금 생성된 캠페인 이벤트 확인
```

⏱️ **소요 시간**: 10초

## 자주 묻는 질문

### Q1: iframe에서도 볼 수 있나요?

**A**: 캘린더가 공개로 설정되어 있거나, iframe 내에서 로그인하면 볼 수 있습니다. 
하지만 **새 탭에서 여는 것을 권장**합니다.

### Q2: 매번 로그인해야 하나요?

**A**: 아니요! 새 탭에서 열면 현재 Google 세션을 사용하므로 자동으로 로그인됩니다.

### Q3: 캘린더를 공개로 설정하면 되지 않나요?

**A**: 보안상 권장하지 않습니다. 광고 일정은 민감한 정보이므로 비공개로 유지하고 
새 탭에서 여는 방식을 사용하세요.

### Q4: 모바일에서도 동일하게 작동하나요?

**A**: 네! 모바일 브라우저에서도 버튼을 클릭하면 새 탭(또는 앱)에서 Google Calendar가 열립니다.

### Q5: 캘린더 이벤트를 VS-AMS에서 직접 수정할 수 있나요?

**A**: 현재는 캠페인 관리에서 캠페인을 수정하면 자동으로 캘린더에 반영됩니다. 
캘린더에서 직접 수정한 내용은 VS-AMS에 반영되지 않습니다.

## 팁과 트릭

### 💡 Tip 1: 북마크 추가

자주 사용한다면 새 탭에서 연 캘린더를 북마크에 추가하세요.

### 💡 Tip 2: 멀티 모니터 활용

듀얼 모니터를 사용한다면:
- 모니터 1: VS-AMS (캠페인 관리)
- 모니터 2: Google Calendar (일정 확인)

### 💡 Tip 3: 캘린더 단축키

Google Calendar에서 사용할 수 있는 단축키:
- `C`: 새 이벤트 생성
- `T`: 오늘로 이동
- `J / K`: 이전/다음 날짜
- `M`: 월간 보기
- `W`: 주간 보기

### 💡 Tip 4: 캘린더 공유

팀원들과 캘린더를 공유하려면:
1. Google Calendar에서 설정 열기
2. 특정 사용자와 공유 선택
3. 팀원 이메일 추가
4. 권한 설정 (보기/수정)

## 문제 해결

### 새 탭에서도 로그인 요구

**원인**: Google 세션이 만료됨
**해결**: Google에 다시 로그인하세요

### 캘린더가 비어 있음

**원인**: 캠페인이 아직 없거나 Google Calendar ID가 잘못됨
**해결**: 
1. 캠페인 관리에서 캠페인 생성
2. 캘린더 ID 확인 (`mj@venturesquare.net`)

### 버튼이 작동하지 않음

**원인**: 팝업 차단
**해결**: 브라우저 팝업 차단 설정 확인
